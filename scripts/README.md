# 手动测试和工具脚本

## 目录说明

这个目录包含手动测试脚本和实用工具脚本。

## 脚本列表

### 手动测试脚本

#### `manual_test_ssh_key.py`
测试SSH私钥文件上传功能的手动测试脚本。

**使用方法：**
```bash
python scripts/manual_test_ssh_key.py
```

**注意：**
- 需要先创建测试用户（linyuan）
- 需要配置好数据库
- 这是手动测试，不是自动化测试

## 脚本规范

### 命名规范
- 手动测试脚本：`manual_test_*.py`
- 工具脚本：`tool_*.py` 或描述性名称
- 数据处理脚本：`process_*.py`

### 脚本要求
1. 每个脚本应该有清晰的文档字符串
2. 包含使用说明
3. 处理错误情况
4. 提供有用的输出信息

## 与自动化测试的区别

- **自动化测试**（`tests/`）：可以在 CI/CD 中运行，不依赖特定环境
- **手动测试**（`scripts/manual_test_*.py`）：需要手动运行，可能依赖特定数据或环境
- **工具脚本**（`scripts/tool_*.py`）：辅助开发的实用工具

## 添加新脚本

添加新脚本时：
1. 遵循命名规范
2. 添加到本文档的脚本列表
3. 包含使用说明和示例
